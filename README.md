# MiniWebProject

# 기술 스택
Front-end: react, typeScript
Back-end: Spring Boot 2.7.6
인증/인가: JWT Token, Spring Security
컨테이너화: Docker                                  
배포: AWS ec2, GitHub Actions을 이용한 ci/cd 자동배포    
데이터베이스: aws mySql                          
ORM : jpa
외부 라이브러리 : Ck Editor, Material UI
파일 저장 : db -> aws S3
